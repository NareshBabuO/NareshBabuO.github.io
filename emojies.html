                         Article about the html

HyperText Markup Language (HTML) is a type of markup language.[1] It is used to make webpages.
Webpages can include writing, links, pictures, and even sound and video.
HTML tells web browsers what webpages should look like. HTML also adds meta information to webpages. Meta information is information about about a webpage e.g.,
the name of the person who created the page. Web browsers usually do not show meta information.
